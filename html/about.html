Sobre a Burger House

A Burger House é um site fictício desenvolvido para representar uma hamburgueria moderna, utilizando HTML, CSS e JavaScript no desenvolvimento front-end.

O projeto foi criado com o objetivo de praticar a construção de páginas web interativas, estilização visual e validação de formulários utilizando apenas tecnologias básicas da web.

O site apresenta um menu de navegação funcional, página inicial com produtos, formulário para contato do usuário e tratamento de dados utilizando JavaScript.
